# dsc_bio — Claude Code Guide

Personal bio/resume site for David Chelimo, hosted on GitHub Pages at `dchelimo.afyagateway.io`.

---

## Stack

| Layer | Tool |
|---|---|
| Styles | Tailwind CSS v3 (compiled via CLI) |
| Fonts | DM Serif Display · DM Sans · JetBrains Mono (Google Fonts) |
| PDF | WeasyPrint 66.0 via `scripts/generate_pdf.py` |
| CI | GitHub Actions — rebuilds CSS + PDF on every push to `main` |
| Hosting | GitHub Pages — custom domain via `CNAME` |

## Local dev workflow

```bash
# Install dependencies (first time only)
npm install
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt

# During development
npm run watch                        # recompile CSS on every file save

# Regenerate PDF locally
npm run build:css
.venv/bin/python scripts/generate_pdf.py

# Deploy — Actions rebuilds CSS + PDF automatically
git add . && git commit -m "..." && git push
```

## PDF print styles (`@media print`)

The same `index.html` renders as a **1-page A4 resume** when printed. Key rules:

- **Hidden in PDF:** site bar, header links, About section, Selected Work, Languages, footer
- **Shown in PDF:** header summary, all 5 experience entries (earlier experience expanded), Skills, Education, Publications
- **Bullets capped at 3 per experience entry** via `.exp-bullets li:nth-child(n+4) { display: none }`
- **Layout:** `display: table` two-column (59% left / 41% right) — floats and CSS Grid are unreliable in WeasyPrint
- **Page:** A4, margins 10mm top/bottom · 13mm left/right
- **Font sizes:** 8.5pt body, 8pt bullets, 22pt name
- Gradient text replaced with solid navy for the name

To regenerate the PDF locally after any content change:
```bash
npm run build:css && .venv/bin/python scripts/generate_pdf.py
```

## GitHub Actions (`build.yml`)

Triggers on push to `main` when any of these paths change: `index.html`, `src/**`, `tailwind.config.js`, `scripts/**`, `package.json`, `package-lock.json`, `requirements.txt`. Changes to `CLAUDE.md`, `CNAME`, `.gitignore`, etc. do not trigger a build.

Steps:
1. Checkout repo
2. Node.js 20 → `npm ci` → `npm run build:css`
3. Python 3.11 → install WeasyPrint system libs (pango, cairo, harfbuzz, fontconfig) → `pip install -r requirements.txt`
4. `python scripts/generate_pdf.py`
5. `git add -f dist/styles.css David-Chelimo-Resume.pdf` (force-add because `dist/` is gitignored)
6. Commit `[skip ci]` + push via `GITHUB_TOKEN`

Requires `permissions: contents: write` on the job.
